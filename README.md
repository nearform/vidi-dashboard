# bg-dashboard

- __Sponsor:__ [nearForm][Sponsor]

A web app used to chart data from the BrewGauge system.

- __Work in progress:__ This module is currently a work in progress.

## Running
To run the dashboard,

1. Run `npm install`
2. Run `npm run build`
3. Run `npm run start`
